# Buzzword Cache

Buzzword Cache for the _Multi-Player Bingo_ game. Returns a map of buzzwords.

##### Based on the course [Multi-Player Bingo](https://pragmaticstudio.com/courses/unpacked-bingo) by Mike and Nicole Clark.
